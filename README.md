
## API

The package is available by importing its default function:

```js
import clone from '@wrote/clone'
```

### `clone(`<br/>&nbsp;&nbsp;`arg1: string,`<br/>&nbsp;&nbsp;`arg2?: boolean,`<br/>`): void`

Call this function to get the result you want.

```js
/* yarn example/ */
import clone from '@wrote/clone'

(async () => {
  await clone()
})()
```
